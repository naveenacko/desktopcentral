ManageEngine Desktop Central Remote Office Setup Wizard - Read Me


To install the Agent in a Single Computer:

  1.  Double-click the setup.bat in the extracted folder. This invokes the installation process.
  2.  Follow the instructions and install the DesktopCentral Agent.

To install Agent in multiple computers of the same domain:
  1. Edit the computernames.txt file in the extracted folder and add all the computer names to which the agent has to be installed. Each computer has to be specified in a separate line.
  2. Execute the setup.bat file and read the instructions as mentioned in the batch file.
  3. Specify the User Name and Password in the respective fields.  
    Note: User Name should be in the format, [domain or workgroup]\[administrator]
               Example:  zohocorp\administrator
     

Refer to the Online Knowledge Base for possible problems during agent installation.
